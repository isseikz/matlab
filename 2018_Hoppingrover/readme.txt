How to use

run HRmain.m under Matlab environment.

Participants are required to provide a file "Control.m".
!!! IMPORTANT !!!
copy "Control.m" (and other matlab programs you use) into the folder ./entry00
!!!!!!!!!!!!!!!!!

The sample program was checked under the following environment.
	OS: Windows 7 Professional
	Matlab: 7.5.0(R2007b)
	Matlab: 7.13.0(R2011b)
	Matlab: 8.5.0(R2015a)
Because no special functions are used, the sample program shall be run under different versions of Matlab.
